# Mage Knight Advanced Action Cards Reference

Source: Card images extracted to `docs/sprites/cards/advanced_actions_full/`

Advanced Actions are powerful cards that can be acquired during the game. Each card has a basic effect (top text box) and a powered/strong effect (bottom text box).

**Total: 44 Advanced Action Cards** (40 base game + 4 Ultimate Edition)

## Scaling Effects

| Card | Scaling Effect |
|------|----------------|
| **Blood Rage** (#04) | +3 Attack if you take a Wound (basic), +5 Attack if you take a Wound (powered) |
| **In Need** (#19) | +1/+2 Influence per Wound in hand and on Units |
| **Counterattack** (#28) | +2/+3 Attack per enemy blocked this turn |
| **Power of Crystals** (#40) | +1/+2 per crystal of matching color in Inventory |
| **Shield Bash** (#31) | Armor -1 per excess block point (powered only) |
