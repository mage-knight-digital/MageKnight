# Mage Knight Spell Cards Reference

Source: Card images extracted to `docs/sprites/cards/spells_full/`

Spells require mana of their color to cast. Each spell has a basic effect (top) and a powered/strong effect (bottom).

**Total: 24 Spell Cards** (6 per color)

**Legend:**
- **●** = Black dot (appears in top-right corner of basic effect)
- Card # = The X/377 number from bottom-right corner

## Notes

- File # corresponds to extracted files: `docs/sprites/cards/spells_full/spell_XX.jpg`
- Cards with ♥ prefix on card number (299, 300, 301, 302) appear to be from an expansion
- **Black dot (●)** appears on 3 cards: Energy Flow (#00), Mana Meltdown (#06), Mind Read (#18) - one per color except blue

## Scaling Effects

| Card | Scaling Effect |

| **Sacrifice** (#11) | Attack per pair of crystals in inventory |
| **Wings of Night** (#23) | +1 Move point cost per additional enemy targeted (1, 2, 3...) |
