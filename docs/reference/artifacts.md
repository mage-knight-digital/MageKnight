# Mage Knight Artifact Cards Reference

Source: Card images extracted to `docs/sprites/cards/artifacts_full/`

Artifacts are powerful items that can be found during exploration or purchased. Each artifact has a basic effect (top text box) and a powered/strong effect (bottom text box).

**Total: 25 Artifact Cards** (24 base game + 1 Ultimate Edition)

**Legend:**
- Card # = The X/377 number from bottom-right corner
- UE = Ultimate Edition card

## Notes

- File # corresponds to extracted files: `docs/sprites/cards/artifacts_full/artifact_XX.jpg`
- Cards 16-23 appear to be from expansions (card numbers 303-310)
- Card 24 is marked "UE" (Ultimate Edition)
- Banners are special artifacts that can be assigned to Units

## Scaling Effects

| Card | Scaling Effect |
|------|----------------|
| **Banner of Glory** (#00) | Fame +1 per Unit that attacks/blocks (powered) |
| **Sword of Justice** (#08) | Attack 3 per card discarded |
| **Horn of Wrath** (#09) | +1 to +5 Siege Attack with risk of Wounds |
| **Bow of Starsdawn** (#16) | Ranged Attack 2 per card discarded |
| **Golden Grail** (#12) | Fame +1 per point of Healing spent (basic) |
| **Soul Harvester** (#19) | Crystal per enemy defeated (powered) |
