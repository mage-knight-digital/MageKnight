# Mage Knight Unit Cards Reference

Source: Card images extracted to `docs/sprites/units/units_regular_full/` and `docs/sprites/units/units_elite_full/`

Units are recruited followers that provide abilities during combat. Each unit has stats and one or more abilities.

**Total: 30 Unit Cards** (15 Regular + 15 Elite)

**Legend:**
- **Cost** = Influence cost to recruit (top-left)
- **Level** = Unit level I-IV (top-right Roman numeral)
- **Armor** = Damage needed to wound (bottom-right number)
- **Site icons** = Where unit can be recruited (castle, village, monastery, etc.)
- Resistances shown as element icons next to armor

## Notes

- File # corresponds to extracted files in the respective `_full` folders
- Card 15 in both sets is the card back
- **Heroes** units (08-11) are special: cost shows "x2" meaning there are 2 copies each, different abilities per card
- Regular units are Level I-II, Elite units are Level III-IV
- Site icons indicate where units can be recruited (village, monastery, keep, mage tower, city)
- Armor value with * indicates special damage assignment rules

## Resistances Key

- Fire = Fire resistance (ignores Fire attacks)
- Ice = Ice resistance (ignores Ice attacks)
- Phys = Physical resistance (ignores non-elemental attacks)

## Unit Ability Icons

- Sword icon = Attack
- Shield icon = Block
- Bow/arrow = Ranged Attack
- Catapult = Siege Attack
- Boot = Move
- Heart/hand = Heal
- Scroll = Influence
- Crystal = Mana/crystal gain
